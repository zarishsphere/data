# ZarishSphere Data & FHIR Resources

This repository hosts the structured data, health standards, and FHIR R5 resources used by the ZarishSphere platform.

## 🏥 FHIR R5 Resources
Located in the `/fhir` directory:
- `CodeSystems`: Bangladesh DGHS, ICD-10, etc.
- `ValueSets`: WHO SMART Guidelines, etc.
- `StructureDefinitions`: Custom profiles for the Rohingya health context.

## 📊 Reference Data
- `facilities.json`: Registry of health facilities in Cox's Bazar.
- `indicators.json`: Standard health indicators for reporting.
